# fastapi-admin-panel

Реализация админки на FastAPI + SqlAlchemy. База данных - PostgreSQL. 

# Инструкция по запуску docker compose 

1. Все команды вводятся в терминале, находясь в рабочей директории проекта, где находятся файлы docker

2. ***docker compose build*** - команда для сборки образа

3. ***docker compose up -d (для запуска в фоновом режиме)*** - после того, как все приготовления завершены, можно запускать контейнер с сервисом и базой данных. Команда миграции моделей в бд и запуск приложения включены в docker-compose файл.
